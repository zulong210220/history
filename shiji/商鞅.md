## 商鞅

```
商君者，卫之诸庶孽公子也，名鞅，姓公孙氏，其祖本姬姓也。鞅少好刑名之学，事魏相公叔座为中庶子。公叔座知其贤，未及进。会座病，魏惠王亲往问病，曰：“公叔病有如不可讳，将柰社稷何？”公叔曰：“座之中庶子公孙鞅，年虽少，有奇才，愿王举国而听之。”王嘿然。王且去，座屏人言曰：“王即不听用鞅，必杀之，无令出境。”王许诺而去。公叔座召鞅谢曰：“今者王问可以为相者，我言若，王色不许我。我方先君后臣，因谓王即弗用鞅，当杀之。王许我。汝可疾去矣，且见禽。”鞅曰：“彼王不能用君之言任臣，又安能用君之言杀臣乎？”卒不去。惠王既去，而谓左右曰：“公叔病甚，悲乎，欲令寡人以国听公孙鞅也，岂不悖哉！”
```

对此事稍有疑问.魏惠王问公叔座谁可继任,公叔座回以商鞅,此事略显蹊跷,王默然,王不说话.王在想什么呢,大概在想公叔座为何推荐商鞅,现在商鞅明显不合适作为相国,为何推荐商鞅呢,而且就算是魏王接纳,朝中大臣也会反对,商鞅又年轻怎么可能坐稳相位呢.怎么说也是推荐一位离相国只有一步之遥的朝中大臣啊,怎么会推荐一个相府的中庶子呢,难道魏国就没有人才吗.即使想要推荐商鞅让其接替相位,也应该提前几年就推荐给魏王啊,怎么现在才推荐呢,是真的想推荐商鞅吗?自己来探病,为何会说出如此不靠谱的话来.从公叔座的角度来说,此时为何会推荐一个不可能接替相位的人呢,一个老丞相难道不知道现在推荐商鞅不合适吗,显然公叔座是知道的,那为何还会这样说,而且旁边还有史官,一种推断是此事为假.

要做董事长起码也得先进董事会,总不能让一个部门经理马上去做ceo吧.

当然还有一种情况,魏王在这之前就已经不信任公叔座了,来探病只是给公叔座一点面子,客气客气.公叔座也知道魏王也只是和他客气一下,自己推荐的人魏王也不会用,商鞅也的确是丞相之才,魏王不信任公叔座,公叔座把商鞅推荐给魏王,魏王也不一定重用商鞅这样的法家人才,当时的法家人才还是收到世袭贵族排挤的,若是魏王不重用不全力支持,商鞅是很难在魏国发展的,故未推荐也就是为及进.公叔座比商鞅年长不少岁数,不用担心商鞅威胁其相位,且其对商鞅有知遇之恩,一路提拔推荐,商鞅对其只有感谢,若是恩将仇报则必然落人口实,因此及时进荐商鞅对公叔座也没有什么坏处,到时候还是自己的人马担任丞相.是故数年不推荐公孙鞅.而后还对魏王说你要举国听之,即全力支持他,既然魏王不信任公叔座,自然对商鞅也不怎么待见,怎么又会举国听之呢,若是魏王很了解信任公孙鞅尚且有可能,公孙鞅一个相府的人,魏王不怎么熟悉,又怎么会全力支持他呢.故公叔座给的建议魏王不会采纳,魏王无所获,故对其所说只话也不会太放在心上.公叔座劝其杀商鞅魏王也不会听.因为公孙鞅与魏王无冤无仇,且对魏王没有任何威胁,无故听人言杀之,恐传出去魏国人人自危.公叔座此间两翻言语确实是尽忠尽义,只是魏王不信任他,只做表面功夫而已.十数年之后才知此是忠言.此亦说明公孙鞅与公叔座交情其实还是很不错的.

公叔座建议魏王杀了公孙鞅说明公叔座认为公孙鞅已经得到了锻炼成了气候,现在不重用他,若是他去了别国得到了重用,可能会对魏国很不利.前面因为魏王对其不信任故没有推荐.此时要用公孙鞅就要不拘一格破格提拔.

```
孝公既用卫鞅，鞅欲变法，恐天下议己。卫鞅曰：“疑行无名，疑事无功。且夫有高人之行者，固见非于世；有独知之虑者，必见敖于民。愚者暗于成事，知者见于未萌。民不可与虑始而可与乐成。论至德者不和于俗，成大功者不谋于众。是以圣人茍可以强国，不法其故；茍可以利民，不循其礼。”孝公曰：“善。”甘龙曰：“不然。圣人不易民而教，知者不变法而治。因民而教，不劳而成功；缘法而治者，吏习而民安之。”卫鞅曰：“龙之所言，世俗之言也。常人安于故俗，学者溺于所闻。以此两者居官守法可也，非所与论于法之外也。三代不同礼而王，五伯不同法而霸。智者作法，愚者制焉；贤者更礼，不肖者拘焉。”杜挚曰：“利不百，不变法；功不十，不易器。法古无过，循礼无邪。”卫鞅曰：“治世不一道，便国不法古。故汤武不循古而王，夏殷不易礼而亡。反古者不可非，而循礼者不足多。”孝公曰：“善。”以卫鞅为左庶长，卒定变法之令。
```


简单描述一下商鞅之死:被告谋反,逃亡,走商邑,兵败身死.

商鞅每次出行都是护卫森严,甚至可以说是数量多余秦君的.商鞅能够从国都一直逃到边境,说明秦惠文君并不是蓄意谋之.商鞅应该是带着手下一路打一路逃到边境.商鞅被退返回秦国之后还能发邑兵说明商邑有很多商鞅的心腹,帮其控制商邑.而后商鞅没有反攻秦国,二是选择攻打郑国,并打进了郑国,而后秦国追兵一路杀到郑国,击败并杀死商鞅.按理说商鞅已经进了郑国,秦国打商鞅有打郑国的嫌疑,那是谁发的兵呢,是刚即位秦惠文君吗,没有明确记载,可能不是,那是谁呢?其实就是那些秦孝公一辈的老贵族,因为他们手里才握有兵权.

秦孝公死前对商鞅什么态度没有记载,但是其封商十五邑,说明其很大方了.有人说商鞅功高震主,其实是后世视角,当时的君主很多,此处待不下去可以去他国.到了秦庄襄王时还与吕不韦还封侯食十万户.秦国封侯的有张仪,魏冉,范雎,吕不韦等等.穰侯还寿终正寝与封邑.其实商鞅变法就是将之前的爵位世袭由血缘改为军功.秦孝公将除掉商鞅收回封邑交给惠文君去做也是不靠谱的,惠文君初立,商鞅大权在握,若是处理不好双方在都城打起来造成内乱怎么办,岂不更糟.

此时应该也有部分能力强的公子选择适应新法,新法对那些无能的老贵族最不利.此时的秦国一部分是老贵族,一部分是新兴贵族,相对平衡.此时秦国需要商鞅这样的人,秦孝公依法给商鞅封侯,基本可以断定秦孝公是没有除掉商鞅收回封邑的想法的,变法让秦国实力快速变强,这点秦孝公感受最明显,秦孝公基本也是没有一统天下的想法的,因为秦国才刚刚复强.秦孝公即位时二十一岁,此时商鞅虽然也年轻,但是显然是比秦孝公年长的.秦孝公的一种可能的想法是,商鞅在朝中稳几年,让惠文君成长起来之后过几年告老还乡回封邑.

商鞅之法是有利于国家的,直接受益人是国君,为君王服务的.秦国现在处在国力上升期,需要这样的人,若是设计除掉商鞅,会让人兔死狐悲,不愿为秦国效力,结果是身死地收的话全是给秦国白干.

总的来说秦孝公是很感谢商鞅的,商十五邑也是给商鞅一个退路.商鞅帮他实现了缪公之业，东复侵地的人生理想,秦孝公也应当兑现其承诺,不然可能重蹈秦缪公的覆辙.秦国强大很大程度上都是任用外来的能人,这些人也都是图功名富贵而来.秦孝公还想用商鞅来镇住老贵族,毕竟商鞅在秦国造反夺取君位是不可能的.而且若有变,其可带着自己的卫队出逃.



秦惠文君是个能力很强的人,其对变法感受也很深,毕竟自己犯法,其师受刑.他也看到了秦法给秦国带来的实实在在的好处.那他是否痛恨商鞅呢,是否会在即位之后对商鞅痛下杀手呢,或者自己不动手借刀杀人呢?
